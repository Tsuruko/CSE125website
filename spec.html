Project Spec

Due April 6, 2015

Project Description

  • Game Type
    • A sandbox building game where the player builds and customizes a battle robot to fight other bots.
  • Game Goals
    • A player wins if his robot is the last one standing, or receives the least amount of damage before a time limit.
  • Game Aspects
    • Build and customize your own bot
    • Different battlefields
    • Gameplay changes based on parts you pick
  • Game Features
    • Must Have
      • Physics Engine
      • Damage System
      • Robot Template(s)
      • Basic Graphics
      • Multiple Cameras
      • Basic Models and art
    • Would Be Really Nice
      • Unique Battlefields
      • Advanced Graphics/Rendering
      • Particle Effects/Special Effects/Water Effects
      • Advanced Physics
      • Fly System
      • Level of Detail for models
    • Cool But Only If Ahead Of Schedule
      • Cosmetic Robot Customization
      • Optimize Connection
      • Level Editor
      • AI System
      • Weather System
      • Art Details

Group Management

  • Roles
    • Everyone will work on everything for management, Democracy!
  • Decisions
    • We will make decisions as a group during our daily meetings. If there is a disagreement or discussion that takes too much time, we will flip a coin or go to an advisor for help.
  • Communication
    • Daily Meetings from 6-9pm
    • Facebook Group
    • Email
  • Scheduling
    • We will know if we are off schedule based on the development plan linked below.
    • At our daily meetings we can decide which tasks have the highest priority and what needs to be changed if we become off-schedule.
  • Weekly Status Reports
    • status reports will be compiled weekly from the daily meetings and reviewed at the beginning of each week.
